Threat Detection in Spoken Amharic Language

[Website]

I created a real-time threat detection system for spoken Amharic, using natural language processing (NLP) to classify audio data into threat and non-threat categories. To achieve this, I utilized various feature extraction techniques, including Mel-frequency cepstral coefficients (MFCCs), Chroma Features, and Spectral Features. I also incorporated data augmentation methods like time stretching, pitch shifting, and noise addition to make the model more robust and adaptable. Throughout the project, I explored different acoustic models, such as Convolutional Neural Networks (CNN), Recurrent Neural Networks (RNN), and Long Short-Term Memory (LSTM) networks, to find the best architecture for accurately identifying threats in spoken language.
